## 第 4 关：修复成绩等级判断

- 分值：3
- 题型：代码改错
- 知识点：if/elif/else、比较运算符、逻辑顺序
- 任务说明：下面代码有错误。请修复后，让每个分数得到正确等级。
- 规则：90 分及以上为 A，80 到 89 为 B，70 到 79 为 C，60 到 69 为 D，60 以下为 E。

初始代码：

```python
scores = [96, 82, 77, 61, 45, 90, 80, 59]
levels = []

for score in scores:
    if score > 90:
        level = 'A'
    elif score > 80:
        level = 'B'
    elif score > 70:
        level = 'C'
    elif score > 60:
        level = 'D'
    else:
        level = 'E'
    levels.append(score)
```

判题变量：

```python
levels = ['A', 'B', 'C', 'D', 'E', 'A', 'B', 'E']
```

## 第 5 关：修复签到统计器

- 分值：3
- 题型：代码改错
- 知识点：for 循环、continue、计数器、列表
- 任务说明：下面代码要统计有效签到。空字符串和 None 不计入有效签到，重复姓名只保留第一次出现。

初始代码：

```python
records = ['林小星', '', '周远航', '林小星', None, '许晨光', '周远航', '沈知秋']

valid_students = []
valid_count = 0

for name in records:
    if name == '' and name == None:
        continue
    if name in valid_students:
        valid_students.append(name)
    valid_count = valid_count + 1
```

判题变量：

```python
valid_students = ['林小星', '周远航', '许晨光', '沈知秋']
valid_count = 4
```

```python

score_b = calc_final(70, 75)
score_c = calc_final(100, 85, 0)
```

## 第 10 关：生成学习接口数据

- 分值：5
- 题型：写大部分代码
- 知识点：import、json 模块、函数、字典、JSON 字符串
- 任务说明：模拟把学习报告发送给服务器。不要真的访问网络，只需要生成接口请求所需的数据。

初始代码：

```python
students = [
    {'name': '林小星', 'score': 92, 'finished': 8},
    {'name': '周远航', 'score': 81, 'finished': 7},
```

- `payload` 为函数返回的字典。

```python

# 2. StudentReport.get_status(self, total_lessons)：
#    finished 等于 total_lessons 且 score >= 85，返回 '优秀完成'
```

## 难度递进说明

1. 第 1 到 3 关只做补全，重点降低上手阻力，考察基础语法和容器概念。

3. 第 7 到 9 关开始写主要逻辑，覆盖循环统计、文件操作、面向对象基础。
4. 第 10 到 12 关做综合任务，逐步加入 JSON、对象系统、报告生成，避免真实联网导致判题不稳定。
