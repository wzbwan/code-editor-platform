# 二、什么是客户端与服务器

访问网页时：

text 浏览器    ↓ 发送请求(Request)    ↓ 互联网    ↓ 服务器(Server)    ↓ 返回响应(Response) 

Python程序也可以充当浏览器。

例如：

python import urllib.request  urllib.request.urlopen("https://www.baidu.com") 

实际上就是：

text Python    ↓ 请求网页    ↓ 百度服务器    ↓ 返回网页内容 

---

# 三、第一个网络程序

python import urllib.request  response = urllib.request.urlopen(     "https://www.baidu.com" )  print(response) 

输出：

python <http.client.HTTPResponse object> 

说明：

- 已成功连接服务器
- 得到了服务器返回的数据

# 四、读取网页内容

服务器返回的数据保存在 response 中。

读取内容：

python import urllib.request  response = urllib.request.urlopen(     "https://www.baidu.com" )  data = response.read()  print(data[:100]) 

输出类似：

python b'<!DOCTYPE html>...' 

注意：

前面的 b 表示：

python bytes 

字节数据。

# 五、字节转字符串

网页内容通常需要转换为字符串。

python import urllib.request  response = urllib.request.urlopen(     "https://www.baidu.com" )  html = response.read().decode("utf-8")  print(html[:300]) 

decode()：

python 字节 → 字符串 

这是网络编程中最常见的操作之一。(Stack Overflow)

# 七、获取JSON数据

例如：

python import urllib.request  url = "https://httpbin.org/get"  response = urllib.request.urlopen(url)  print(     response.read().decode() )

text 获取数据 读取网页 调用接口

# 十、发送POST请求

很多API要求POST请求。

步骤：

## 第一步：准备数据

python import json  data = {     "name":"Tom",     "age":18 } 

---

## 第二步：转换为字节

python body = json.dumps(data)  body = body.encode("utf-8") 

---

## 第三步：创建Request对象

python import urllib.request  req = urllib.request.Request(     url="https://httpbin.org/post",     data=body,     method="POST" ) 

---

## 第四步：发送请求

python response = urllib.request.urlopen(req)  print(     response.read().decode() ) 

Request对象用于描述一个HTTP请求。(Python documentation)

---

# 十一、添加请求头

有些网站要求：

text User-Agent Authorization Content-Type 

添加方式：

python headers = {     "Content-Type":"application/json" } 

创建请求：

python req = urllib.request.Request(     url=url,     data=body,     headers=headers,     method="POST" ) 

---

# 十二、调用AI接口

大语言模型API本质上也是网络请求。

流程：

text Python    ↓ 发送POST请求    ↓ AI服务器    ↓ 返回JSON    ↓ 提取回答 

示例：

python import urllib.request import json  url = "API地址"  headers = {     "Authorization":"Bearer API_KEY",     "Content-Type":"application/json" }  data = {     "model":"deepseek-chat",     "messages":[         {             "role":"user",             "content":"你好"         }     ] }  body = json.dumps(data).encode()  req = urllib.request.Request(     url=url,     data=body,     headers=headers,     method="POST" )  response = urllib.request.urlopen(req)  result = json.loads(     response.read().decode() )  print(     result["choices"][0]["message"]["content"] ) 

---

# 十三、异常处理

网络请求可能失败。

例如：

- 没有网络
- 地址错误
- 服务器故障

建议：

python import urllib.request import urllib.error  try:      response = urllib.request.urlopen(         "https://abcxyz123.com",         timeout=5     )  except urllib.error.URLError:      print("网络访问失败") 

官方文档中 urllib.error 模块专门用于处理网络异常。(Python documentation)
